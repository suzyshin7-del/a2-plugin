Mood Blob

A playful interactive character built with Pts.js.
Move your cursor and click to explore its changing moods.

Move your pointer to interact with the blob. Click on the canvas to change its mood.

Design Rationale

I chose Pts.js for this proof-of-concept because it offers a lightweight and flexible way to create interactive visual experiences directly in the browser. I was interested in exploring how a simple geometric character could become expressive through motion and pointer-based interaction. Using Pts.js made it possible to connect user input with real-time changes in shape, facial expression, and movement.

For this project, I created a playful blob character whose eyes, mouth, and body respond to the user’s cursor. This approach demonstrates how JavaScript can be used not only for functionality, but also for personality and engagement. Beyond this proof-of-concept, similar interactions could be used in website mascots, onboarding screens, educational tools, or storytelling interfaces that aim to feel more dynamic and emotionally responsive.
